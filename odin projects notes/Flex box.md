## Flexbox: 
- a way to arrange items into rows or columns
- items grow or shrink
	- a flex will fill an available area and the container will have equal width
	- ==Flex container==  any element that has (`display: flex`) on it.
	- ==Flex item:== any element that lives directly inside of a flex container

		- any element can be a flex container and item.
