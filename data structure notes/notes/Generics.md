what do they do?
- ==Generic data type== generics enable you to write a placeholder instead of an actual class type within the definition of a class or interface
	- `public class OrderedPair<T> implements Pairable<T>`
- to establish a generic type when defining an interface/class, write an identifier like `<T>` can be any identifier but is usually a single capital letter 
	- Class: `orderedPair`
